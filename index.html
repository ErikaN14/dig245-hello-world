Tutorial: Data Types

The tutorial that I completed focused on the different data types that exist. The tutorial starts by discussing these different data types and splitting them into two groups, primitive and objects. The tutorial includes one section that gives examples of the different primitive types and objects being discussed. With each example,there are also practice activities that allow you to see the output of each using the console. There is also part of the tutorial that focuses on the 'typeof' command which is useful when identifying what data type you are working with. Finally, the tutorial goes into detail on two data exchange formats, Tabular Data and Hierarchical Data. I thought that the various examples included for each of these gave me a better understanding of the differences between the two. After completing this tutorial, I feel like I have a better understanding of what different data types there are.
